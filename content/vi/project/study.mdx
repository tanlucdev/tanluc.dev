### Về Dự Án
**Study Stream** là một ứng dụng web full-stack hỗ trợ người học quản lý sự tập trung, thời gian và động lực. Được phát triển như một phần của dự án luận văn, nó tích hợp kỹ thuật Pomodoro với các công nghệ full-stack hiện đại.

### 🔧 Tính Năng Chính

- ⏲️ **Bộ Hẹn Giờ Pomodoro**: Các phiên tùy chỉnh với nền âm thanh, nhạc và danh sách công việc.
- 💬 **Chat Thời Gian Thực & Trợ Lý AI**: Nhắn tin với bạn bè cùng chatbot năng suất (WebSocket + Agora).
- 📚 **Không Gian Học Tập**: Khám phá và đăng ký các khóa học theo chủ đề.
- 📊 **Phân Tích Tập Trung**: Theo dõi thời gian học hàng ngày, trung bình và tổng cộng.
- 👤 **Hồ Sơ Người Dùng**: Cập nhật avatar, theo dõi thành tích và lịch sử.

### Điểm Nổi Bật Tech Stack

- **Frontend**: React với Tailwind CSS và MUI cho UI mô-đun, responsive.
- **Backend**: Node.js, Express, MongoDB (kiến trúc MERN) với xác thực JWT.
- **Thời Gian Thực**: WebSocket + Agora API cho chat và tương tác trực tiếp.
- **DevOps**: Frontend trên Vercel, backend trên Render.

### Những Gì Nó Thể Hiện

- Nắm vững phát triển full-stack sử dụng các công cụ JS hiện đại.
- Kinh nghiệm với API thời gian thực và tích hợp dịch vụ bên thứ ba.
- Kiến trúc sạch với tách biệt mối quan tâm, component tái sử dụng và thiết kế API RESTful.
- Tập trung vào trải nghiệm người dùng + hiểu biết dữ liệu thông qua UI tương tác và phân tích.
